## Compatibilidad actual

- El motor sigue siendo `PhotoOrganizer.ps1` y vive en `App\PhotoOrganizer.ps1`.
- El lanzador sigue siendo `Start-PhotoOrganizer.cmd`.
- La referencia de comandos sigue siendo `Docs\CommandReference.html`.
- No se renombran JSON, logs, carpetas internas, aliases, runtime, scripts ni configuraciones por branding.
- El branding visual no cambia nombres tecnicos, rutas, parametros ni compatibilidad.
- Para uso real consulta `README.md`, `Docs\Manuals\Manual_ES.md`, `Docs\Manuals\Manual_RO.md` y `Docs\CommandReference.html`.

## Regla de proyecto

```text
branding visual
!=
nombres tecnicos internos
```

UDMRS Dashboard es la identidad visual del producto. PhotoOrganizer sigue siendo el nombre tecnico del motor PowerShell.
