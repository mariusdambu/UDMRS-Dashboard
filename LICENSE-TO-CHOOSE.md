# License Required Before Public Release

This prepared GitHub package intentionally does not choose a legal license for the project owner.

Before publishing publicly, add a `LICENSE` file.

Common options:

- MIT: permissive, simple, allows reuse with attribution.
- Apache-2.0: permissive, includes explicit patent language.
- GPL-3.0: copyleft, requires derivative works to remain GPL-compatible.
- All rights reserved: publish source for visibility but do not grant reuse rights.

Until a license is added, assume no permission is granted beyond viewing the files.
