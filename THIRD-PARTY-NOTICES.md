# Third-party Notices

## ExifTool

UDMRS can use ExifTool for metadata reading and EXIF repair workflows.

ExifTool is not bundled in this public source package.

Project:

```text
https://exiftool.org/
```

Users should install ExifTool themselves and review its license.

Expected local placement:

```text
Tools\ExifTool
```

## PowerShell

UDMRS runs on Windows PowerShell 5.1 or PowerShell 7+.
