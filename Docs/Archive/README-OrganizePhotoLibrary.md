# Organize-PhotoLibrary.ps1

> Documento legado. No es la guía del dashboard actual.

Este archivo queda solo como referencia histórica del script antiguo `Organize-PhotoLibrary.ps1`.

Para el flujo actual usa:

```text
Start-PhotoOrganizer.cmd
Manual_ES.md
Manual_RO.md
PhotoOrganizer.ps1
```

El dashboard actual usa `PhotoOrganizer.ps1`, perfil `QuarterlyFolders`, reconciliación automática post-Normalize, índice incremental en `%APPDATA%\PhotoOrganizer\` y backups de metadatos en `%LOCALAPPDATA%\PhotoOrganizer\`.

No uses este documento para decidir carpetas, comandos ni flujo de trabajo actual.
